Movement.h

Heads-up for non-team members

This page is a reference meant for Axis of Innovation members. While you can still access it, it doesn't exactly serve much use as this is documentation for a file specific to our current robot only.

File Description

This is the file we use for handling all the low-level movement with the robot, making it easier to read the written code and much, much faster to write, modify, and debug the code as it's being written.

Method Overview

void move( int left, int right )
Sets the designated powers to the correct wheels.

Parameter Description
int left The new power for the left wheel
int right The new power for the right wheel


void move( int power )
Sets the designated power to both the robot's wheels.
This causes either completely straight (either forward or backward) movement with no programatic possibility of rotation.

Parameter Description
int power The new power for both wheels.


void brake()
Stops the robot's wheels by calling move( 0, 0 )


void moveEncoder( int ticks, int power )
Moves the robot straight (either forwards or backwards) until the specifid motor encoder value has been met.

Parameter Description
int ticks The number of ticks to move towards based off of the right motor's encoder readings)
int power The power to move the robot straight with.

void moveDistance( int distance, int power )
Moves the robot forward until it's approximately at the given distane.
This will get more inaccurate the further it goes, due to the nature of how the encoder values were calculated, so try for smaller, more gradual steps if you notice any inconsistencies.

Parameter Description
int distance The distance to move the robot (cm).
int power The power to move the robot at.

void rotate( int power )
Rotates the robot at the given speed. Positive is counter-clockwise.

Parameter Description
int power The power to rotate the robot at.

void rotateEncoder( int ticks, int power )
The rotational counterpart for moveEncoder( int, int ), rotates the robot until it has satisified the encoder values.

Parameter Description
int ticks The number of ticks to rotate (positive is counter clockwise).
int power The power to rotate the robot at.

void rotateDegrees( int degrees, int power )
Rotates the robot to the specified degree position, the rotational counterpart for moveDistance( int, int )
The degrees position is defined such that rotating 90 + 180x (where x is an integer) will always only rotate 90Â° in the counter-clockwise direction. This method rotates the the position that the specified number of degrees of rotation would land it at you should not confuse this with rotating the number of specified degrees!

Parameter Description
int degrees The number of degrees to rotate the robot (reduced to the interval [-180°,180°])
int power The power to rotate the robot at.
