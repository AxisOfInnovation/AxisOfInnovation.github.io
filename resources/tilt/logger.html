Logger.h

File Description

This header allows for easier access to the I/O systems in the NXT file system and can log various data types into a human readable text document.

Only members which are meant to be accessed, called, or started in the cases of fields, methods, and tasks, respectively, are shown, some members may be excluded.

Field Overview

DeclarationDescription
bool fileOpenTrue if and only if there is a file currently opened for editing.

Method Overview

void closeLog()
Closes the currently opened log file.
Checks to make sure that there is an opened log file (avoiding potential errors) and if there is one opened, closes it and sets fileOpened to false.


void openLog( string const fileName, int fileSize )
Opens a new log file for writing with a determined file size.
Checks for a previously opened file, and closes it if one is found. Then proceeds to delete the previous file by the same name (nothing will happen if said file does not exist) and then opens the newly created file for writing.

ParameterDescription
string const fileNameThe name of the file to create.
int fileSizeThe size of the file to create (measured in bytes.)


void write( string const s )
Writes a string to the currently opened log file.

ParameterDescription
string const sThe string to write to the file


void write( int i )
Writes a numeric value to the currently opened log file.

ParameterDescription
int iThe numeric value to write to the file


void writeLine()
Writes a line feed character to the currently opened log file.


void writeLine( string const s )
Writes a string to the currently opened log file, then a line feed character.

ParameterDescription
string const sThe string to write to the file


void writeLine( int i )
Writes a numeric value to the currently opened log file, then a line feed character.

ParameterDescription
int iThe numeric value to write to the file
