Sonar.h

File Description

This header file handles all sonar data.

By using the sonarMinimum and sonarMaximum variables, the minmum and maximum values for sonar readings can be changed. When the number violates these boundaries, the closest boundary will be assigned to sonarValue.

Only members which are meant to be accessed, called, or started in the cases of fields, methods, and tasks, respectively, are shown, some members may be excluded.

Field Overview

DeclarationDescription
int sonarValueThe last updated value of the sonar sensor
int sonarMinimumThe minimum value the sonar will read, if a number is below this the sonar will read this instead
int sonarMaximumThe maximum value the sonar will read, if a number is over this the sonar will read this instaed

Method Overview

int updateSonar( tSensors sensor )
Updates and returns the distance the sonar sensor is reading.
Updates the value of the sonarValue global variable and then returns the value of said variable. If the sonar value is out of the [sonarMinimum, sonarMaximum] range, then it will set the value to the closest boundary.

ParameterDescription
tSensors sensorThe sonar sensor's port or name
Returns
The updated value of the sonar sensor
