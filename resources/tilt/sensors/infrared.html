Infrared.h

This header file handles all infrared sensor data.

The infrared sensor has two methods of reading input. The first one is using the direct mode; however, the more usefule one is the zone method. there are five zones, for the direction in which each reads in, please consult the paper shipped with your IRv2 sensor unit. This file offers many ways to reduce annoyances in reading infrared values via rounding or ignoring values below a certain threshold. Both of these functions are useful in filtering out any possible noise the sensor may pick up on; however, it should be noted that these alone will not filter out all noise.

Only members which are meant to be accessed, called, or started in the cases of fields, methods, and tasks, respectively, are shown, some members may be excluded.

int irRoundingAll IR readings will be rounded to the nearest multiple of this number. If the value of this is lesss than or equal to one will be ignored and the values not rounded.

Method Overview

bool initInfrared( tSensors infrared )
Initializes the passed infrared sensor to read for an F.T.C. competition.
This will attempt to initialize the IR sensor to read a 1200 Hertz infrared signal ten times. Each attempt will be followed by 100 milliseconds of waiting time before the next attempt. Upon a tenth failure, the NXT will play the exception sound.

ParameterDescription
tSensors infraredThe ifnrared sensor to initialize.
Returns
true if the initialization of the sensor succeeded, false if it did not.

void updateInfrared( tSensors infrared )
Reads and updates zone readings from the given infrared sensor.
Reads the direct and zone values fromt he ifnrared seeker. Then if a zone reading is less than the threshold sets the zone reading to zero. After checking to ensure that the rounding number is greater than one, each zone is rounded to the enartest multiple of irRounding
